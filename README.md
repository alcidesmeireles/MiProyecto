## Instrucciones para probar

1. Clonar el repositorio:
   
   git clone https://github.com/alcidesmeireles/MiProyecto.git

3 Migrar base de datos:

    python manage.py migrate

3 Ejecutar el servidor:

    python manage.py runserver
